### 安装
    git --version
    
### 初始化
    git config --global user.name xxx
    git config --global user.email xxx
    git config --list

### 区域
    工作区
    暂存区
    版本库

### 对象
    git对象
        key:val 组成的键值对
    树对象
    提交对象

git高层命令
    git init            初始化仓库
    git status          查看文件状态
    git add ./          将修改添加到暂存区
    git commit -m 注释    将暂存区提交到版本库
